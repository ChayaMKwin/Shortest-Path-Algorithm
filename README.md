# Graph Shortest Path Finder

This Python script finds the shortest path between two nodes in a weighted directed graph.

```
    A --5--> B
   |       / | \
   3     1  11   2
   |   /    |     \
   v v      v       v
   C --1--> D --3--> F
    \         /
     5       9
      \     /
       v   v
         E
```
